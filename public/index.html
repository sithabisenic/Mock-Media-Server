Mock Media Server API

Welcome! This API serves Movies, Series, and Songs data.

Available Endpoints:

  • GET /movies, /series, /songs - Retrieve data
  • POST /movies, /series, /songs - Add new item
  • PUT /movies, /series, /songs - Update item (send full object with existing ID)
  • DELETE /movies, /series, /songs - Delete item (send JSON with { id })

Example Data Format (POST/PUT)

{
  "title": "Example Title",
  "year": 2024,
  "artist": "Some Artist"
}
